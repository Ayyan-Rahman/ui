# Frontend Libraries

The main focus of this section is to list the libraries that are available in the frontend.

## Web3 libs
- [Ether.js](https://ethers.org/) - A library for interacting with Ethereum blockchain

## API libs
- [React Query](https://react-query.tanstack.com/) - A library for fetching data from the backend and handling caching.
- [GraphQL-Codegen](https://www.graphql-code-generator.com/) - Schema generator for GraphQL that enables autocomplete and code generation

## UI libs
- [MUI](https://material-ui.com/) - Our base UI library.
- [Framer Motion](https://framer.com/motion) - A library for creating animations in React.
- [react-easy-crop](https://www.npmjs.com/package/react-easy-crop) - A library for cropping images in React.

## React libs
- [React-Use](https://github.com/streamich/react-use) - Collection of React hooks for common use cases.
- [React-Hook-Forms](https://react-hook-form.com/) - A library for creating forms in React.
- [Lexical](https://playground.lexical.dev/) - Text editor

## Util libs

- [clsx](https://github.com/lukeed/clsx) - A library for creating class names.
